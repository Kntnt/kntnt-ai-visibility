Read before writing or editing any Markdown in the repo.

All prose docs – `README.md`, `AGENTS.md`, `CONTRIBUTING.md`, `CHANGELOG.md`, `CONTEXT.md`, everything under `docs/` and the files in `agents.d/` – use British English per `kntnt-text-skills:writing-rules en_GB`: spaced en-dashes ( – ), `-ise`/`-isation` spellings, no Oxford comma, plus the rest of the en_GB mechanics.

Markdown layout: one physical line per paragraph – never hard-wrap prose at a column (wrap newlines pollute diffs and show in some renderers). Blank line between paragraphs; one line per list item. Never reflow fenced code, table rows, blockquotes or YAML frontmatter.
